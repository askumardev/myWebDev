Hyper Text Markup Language

HTML is NOT CASE-SENSITIVE.
HTML contains
1) paired Tags
2) Stand alone Tags
Paired Tags: They contain Starting Tag and also Ending tag
Stand alone tags: They contain only starting tag . No need of ending tag.
Procedure:
1. Open notepad, and type the 'code' and save it with .html extension.
2. Open browser, Select 'open' from file menu, and click on 'browse' and choose the correct path. Then click on 'ok'.
