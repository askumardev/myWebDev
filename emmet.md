# 🚀 Emmet Cheat Sheet

Emmet is a powerful toolkit that helps you write HTML and CSS faster using abbreviations.

---

## 🔹 1. Basic HTML Structure

| Emmet    | Output                  |
| -------- | ----------------------- |
| `!`      | Basic HTML5 boilerplate |
| `html:5` | HTML5 template          |

---

## 🔹 2. Elements

| Emmet | Output        |
| ----- | ------------- |
| `div` | `<div></div>` |
| `p`   | `<p></p>`     |
| `h1`  | `<h1></h1>`   |
| `ul`  | `<ul></ul>`   |
| `li`  | `<li></li>`   |

---

## 🔹 3. Child (`>`)

| Emmet   | Output               |
| ------- | -------------------- |
| `div>p` | `<div><p></p></div>` |
| `ul>li` | `<ul><li></li></ul>` |

---

## 🔹 4. Sibling (`+`)

| Emmet      | Output                        |
| ---------- | ----------------------------- |
| `div+p`    | `<div></div><p></p>`          |
| `h1+p+div` | `<h1></h1><p></p><div></div>` |

---

## 🔹 5. Multiplication (`*`)

| Emmet   | Output             |
| ------- | ------------------ |
| `li*3`  | 3 `<li>` elements  |
| `div*5` | 5 `<div>` elements |
